## ğŸš€ How to Compile and Run

### Prerequisites

- A C++ compiler (e.g., `g++`)
- Standard C++ library support

### Compilation

Navigate to the specific day and part folder, then compile:

```bash
# Example for Day 1, Part 1
cd Day1/part1
g++ -o day1part1 day1part1.cpp
```

### Execution

After compilation, run the executable:

```bash
./day1part1
```

**Note:** Make sure your input file (e.g., `d1p1.txt`) is in the same directory as the executable, or update the file path in the source code accordingly.
